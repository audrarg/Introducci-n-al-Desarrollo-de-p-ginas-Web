Semana 3: Sintáxis HTML5 y CSS3

A continución se presentarán elementos importantes básicos para la construcción de HTML estándar y HTML5. Además se listan propiedades significativas para lograr efectos de estilo en los elementos de HTML por medio de CSS3.

HTML Estándar

div

Define una división o sección en un documento HTML.
Sintáxis:

< div >


Ejemplo:
Esto es parte del div

span

Se usa para poner color a una parte del texto.
Sintáxis:

< span >


Ejemplo:
Ese carro es azul

p

Define un párrafo. Los navegadores agregar márgenes antes de este tag. Sintáxis:

< p >

Ejemplo:

Esto es otro párrafo.

img

Define una imagen en la página HTML.
Sintáxis:

< img src="fuente" >

Ejemplo:

ol

Define una lista ordenada.
Sintáxis:

< ol > (Lista)
< li > (Elemento)

  2. Elemento 2

ul

Define una lista no ordenada.
Sintáxis:

< ul > (Lista)

  • Elemento 2

CSS3

border-radius

Esta propiedad redondea los bordes de los elementos.
Rango de valores:

[ <length> | <percentage> ]{1,4} [ / [ <length> | <percentage> ]{1,4} ]

Ejemplo:
Border radius

box-shadow

Esta propiedad permite agregar efectos de sombra a los elementos.
Rango de valores:

<horizontal_offset> <vertical_offset> <blur> <spread_distance> <color>

Ejemplo:
Box Shadow

text-decoration

Esta propiedad permite agregar efectos de sombra a los elementos.
Rango de valores:

<text-decoration-line> <text-decoration-style> <text-decoration-color>

Ejemplo:

Text Decoration

text-stroke

Esta propiedad permite trazar un texto con bordes más anchos.
Rango de valores:

<border-size> <text-color>

Ejemplo:

Text Stroke

word-break

Especifica oportunidades de ajuste suave dentro de las palabras donde es normal y permisible romper líneas de texto.
Rango de valores:

normal | keep-all | break-all

word-break:

Collaboratively administrate empowered markets via plug-and-play networks.

word-wrap

Especifica si las palabras irrompibles se pueden romper de manera arbitraria para evitar el desbordamiento.
Rango de valores:

normal | break-word

Ejemplo:

Floccinaucinihilipilification of antidisestablishmentarianism.

text-overflow

Especifica que hacer con el texto si es muy largo para el espacio destinado.
Rango de valores:

[ clip | ellipsis | <string> ]{1,2}

Ejemplo:

Este es un texto bastante largo que se va a cortar

Este es un texto largo que no va a ser cortado

Es una propiedad para definir las transiciones de los elementos.
Rango de valores:

'transition-duration' | 'transition-property' | 'transition-timing-function' | 'transition-delay'
